# Wellness Hydration Bar and Spa

A modern, elegant wellness spa website built with Next.js 15, featuring smooth animations and an immersive user experience.

## Features

- **Modern Design**: Clean, spa-inspired aesthetic with calming sage green color palette
- **Smooth Animations**: GSAP-powered animations and transitions
- **Responsive**: Fully responsive design that works on all devices
- **Fast Performance**: Optimized images and assets for quick loading
- **Dynamic Content**: Interactive galleries, spotlights, and client reviews

## Tech Stack

- **Framework**: Next.js 15.4.6
- **React**: 19.1.0
- **Animations**: GSAP 3.13.0 with ScrollTrigger
- **Smooth Scrolling**: Lenis 1.3.8
- **Icons**: React Icons 5.5.0
- **Styling**: CSS Modules with custom properties

## Services Featured

- **Relaxing Bath** - Luxurious bath experiences
- **Sauna** - Traditional and infrared sauna sessions
- **Manicure** - Professional nail care services
- **Jacuzzi** - Therapeutic hot tub sessions
- **Massage** - Various massage therapy options

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm, yarn, or pnpm package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Ubushisas/website-saris-1.git
cd website-saris-1
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm run start
```
